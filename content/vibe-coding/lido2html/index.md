title: "LIDO XML zu HTML Konverter"
date: 2026-05-23T15:33:09+02:00
---

Auch wenn es 2026 gar kein Vibe Coding mehr gibt, hier ist der erste Eintrag in dieser neuen Kategorie.

Mit diesem kleinen Werkzeug können Sie LIDO-XML-Dateien, die Sie beispielsweise aus dem Sammlungsportal der Universität Göttingen exportiert haben, in HTML konvertieren. Im nächsten Schritt können die erzeugten Dateien z.B. im [Arcana/RAG System der GWDG](https://docs.hpc.gwdg.de/services/ai-services/arcana/index.html) weiterverarbeitet zu werden.

Der Code ist bei GitHub unter [`cmahnke/lido2html`](https://github.com/cmahnke/lido2html) verfügbar.
